Жизненный цикл сборки:
1 Clean
Удаляет папку build
2 compileJava
Создает папку build. Компилирует Java-код без запуска тестов. В папке build появляются папки: classes (c откомпилированными классами), generated, tmp.
3 test
Компилирует код и тесты, запускает тесты. Создает build/test-results и build/reports/tests.
4 build
Компилирует основной код и тесты, запускает тесты и создает Создаёт JAR-файл в build/libs
